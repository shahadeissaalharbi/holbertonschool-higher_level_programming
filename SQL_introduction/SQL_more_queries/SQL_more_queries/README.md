# SQL More Queries

This project is a continuation of SQL introduction, focusing on more advanced
MySQL queries: user privileges, joins, subqueries, and views.

## Requirements

- All scripts will run on Ubuntu 20.04 LTS using MySQL 8.0
- All SQL keywords should be in uppercase
- The database name should be passed as an argument of the `mysql` command
- Each file should start with a comment describing the task

## Tasks

### 0. My privileges!
Lists all privileges of the MySQL users `user_0d_1` and `user_0d_2` on `localhost`.

File: `0-privileges.sql`
