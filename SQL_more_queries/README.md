# SQL More Queries

This project covers more advanced MySQL queries: user privileges, joins,
subqueries, and views.

## Requirements

- All scripts run on Ubuntu 20.04 LTS using MySQL 8.0
- All SQL keywords should be in uppercase
- Each file starts with a comment describing the task

## Tasks

### 0. My privileges!
Lists all privileges of the MySQL users `user_0d_1` and `user_0d_2` on `localhost`.

File: `0-privileges.sql`
